# Lecture Script: Data Analysis with Spreadsheets
> **Instructor Reference** — Module 1: Foundations of Data | Session 16 | Duration: 2 Hours

---

## Session Overview

**Goal:** Students use Excel/Sheets for VLOOKUP/XLOOKUP, pivot tables, COUNTIF, and conditional formatting — as a complement to Python/SQL.

**Key outcome:** Build a one-sheet sales dashboard from two related tables without code.
